# SCIE 논문용 연구 질문 정리

## 논문 방향

본 연구는 협동 로봇 실습 매뉴얼을 기반으로, 학습자의 실습 질의에 대해 텍스트 설명과 도식/이미지 정보를 함께 제공하는 상황 인지형 멀티모달 RAG 시스템을 제안한다.

핵심은 단순히 시스템을 구현했다는 점이 아니라, 텍스트 기반 RAG와 비교했을 때 멀티모달 정보와 실습 단계 정보가 검색 성능 및 응답 품질에 어떤 영향을 주는지를 실험적으로 검증하는 것이다.

## 제목 후보

### SCIE 논문 제목 후보

1. A Context-Aware Multimodal RAG Framework for Collaborative Robot Training Guidance
2. Improving Collaborative Robot Training Guidance Using Context-Aware Multimodal Retrieval-Augmented Generation
3. Evaluation of Context-Aware Multimodal RAG for Collaborative Robot Training Manuals

### 국문 석사논문 제목 후보

1. 상황 인지형 멀티모달 RAG 기반 협동 로봇 실습 가이드 연구
2. 협동 로봇 실습 교육을 위한 상황 인지형 멀티모달 검색 증강 생성 시스템 연구

## 핵심 연구 목적

협동 로봇 실습 교육에서 학습자가 매뉴얼을 직접 탐색하지 않고도, 현재 실습 상황에 맞는 설명 텍스트와 관련 도식/이미지를 함께 제공받을 수 있는 오프라인 RAG 기반 실습 보조 시스템을 구현하고 평가한다.

## 연구 질문

### RQ1. 협동 로봇 실습 매뉴얼 기반 RAG 시스템은 실습 질의에 필요한 텍스트 정보를 정확히 검색하고 답변할 수 있는가?

- 목적: 로컬 LLM 기반 텍스트 응답 성능을 확인한다.
- 비교 대상: Qwen, Gemma, Llama 기반 텍스트 응답
- 평가 자료: 70개 질의셋의 정답 텍스트
- 평가 방법: 사람이 직접 텍스트 답변을 O/△/X로 평가

### RQ2. 텍스트 기반 RAG와 비교했을 때, 이미지/도식 정보를 함께 활용하는 멀티모달 RAG는 관련 시각 자료 검색 성능을 개선하는가?

- 목적: 이미지/도식 검색이 실습 안내에 기여하는지 확인한다.
- 비교 대상:
  - 텍스트 기반 RAG
  - 텍스트 + 이미지/도식 기반 멀티모달 RAG
- 평가 자료: 70개 질의셋의 정답 이미지
- 평가 방법:
  - Recall@1
  - Recall@5
  - Recall@10
  - MRR

### RQ3. 실습 단계 정보를 반영한 상황 인지형 검색은 일반 멀티모달 검색보다 실습 맥락에 더 적합한 결과를 제공하는가?

- 목적: 실습 단계 라벨이 검색 범위를 좁히고 응답 맥락을 개선하는지 확인한다.
- 비교 대상:
  - 일반 멀티모달 RAG
  - 실습 단계 라벨을 반영한 상황 인지형 멀티모달 RAG
- 평가 자료: 70개 질의셋의 실습 단계 라벨
- 평가 방법:
  - 실습 단계 적합성
  - 검색 결과의 정답 포함 여부
  - 응답이 해당 실습 상황에 맞는지 수동 평가

### RQ4. 제한된 로컬 환경에서 4-bit 양자화 LLM을 활용한 협동 로봇 실습 가이드 시스템은 교육 보조 도구로 활용 가능한가?

- 목적: 고성능 클라우드 모델 없이도 실습 보조 시스템을 구성할 수 있는지 확인한다.
- 비교 대상:
  - Qwen 2.5 7B Q4
  - Gemma 2 9B Q4
  - Llama 3.1 8B Q4
- 평가 항목:
  - 응답 정확성
  - 안전성
  - 구체성
  - 이해 용이성
  - 로컬 실행 가능성

## 연구 질문과 실험 산출물 연결

| 연구 질문 | 필요한 산출물 | 현재 상태 |
|---|---|---|
| RQ1 텍스트 응답 성능 | 모델별 텍스트 답변 평가표 | 수동 평가용 엑셀 생성 완료 |
| RQ2 이미지 검색 성능 | 정답 이미지, 검색 이미지 Top-k, Recall@k, MRR | 이미지 평가 엑셀 생성 완료, 70개 기준 재평가 필요 |
| RQ3 상황 정보 효과 | 실습 단계 라벨, 상황 반영 검색 결과 | 70개 질의셋 라벨링 완료, 상황 반영 검색 실험 필요 |
| RQ4 로컬 모델 활용 가능성 | Qwen/Gemma/Llama 비교 결과 | 모델 실행 구조 완료, 수동 평가 필요 |

## 현재 연구 범위

이번 SCIE 논문에서는 다음 범위에 집중한다.

1. 협동 로봇 실습 매뉴얼 기반 데이터셋 구성
2. 텍스트와 이미지/도식 자료의 전처리 및 인덱싱
3. 텍스트 기반 RAG와 멀티모달 RAG 비교
4. 실습 단계 라벨을 활용한 상황 인지형 검색 가능성 평가
5. 로컬 4-bit LLM 기반 응답 품질 비교
