# 2차 산출물: 질의셋 70개 확장

## 목적

SCIE 논문용 본 실험에서는 기존 38개 질문만으로는 표본 수가 부족할 수 있으므로, 총 70개 질의셋을 목표로 32개 질문을 추가했다.

## 확장 기준

추가 질문은 다음 기준으로 만들었다.

1. 질문에 대응하는 정답 이미지가 명확할 것
2. `data/processed/final_refined_data` 폴더에 실제 이미지 파일이 존재할 것
3. 기존 38개 정답 이미지와 최대한 겹치지 않을 것
4. 단순 텍스트 답변뿐 아니라 도식, 화면 구성, 배선도, 절차 그림과 연결될 것
5. 실습 상황 또는 실습 단계 라벨을 부여할 수 있을 것

## 산출 파일

| 파일 | 설명 |
|---|---|
| `data/02_additional_questions_32.csv` | 새로 추가한 32개 질문 원본 데이터 |
| `excel/02_additional_questions_32.xlsx` | 한글 확인용 엑셀 파일 |
| `data/03_question_set_70.csv` | 기존 38개 + 추가 32개를 합친 전체 질의셋 |
| `excel/03_question_set_70.xlsx` | 전체 70개 질의셋 확인용 엑셀 파일 |

## 검증 결과

| 항목 | 결과 |
|---|---:|
| 기존 질문 수 | 38 |
| 추가 질문 수 | 32 |
| 전체 질문 수 | 70 |
| 추가 질문 정답 이미지 존재 여부 | 32/32 |
| 전체 질의셋 정답 이미지 존재 여부 | 70/70 |
