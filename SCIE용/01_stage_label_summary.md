# 1차 산출물: 질의셋 실습 단계 라벨링

## 목적

교수님 피드백의 "정답 텍스트, 정답 이미지, 정답 실습 단계 라벨링 기준"에 대응하기 위해 현재 평가 질의셋 38개에 실습 단계 라벨을 부여했다.

## 정리 방식

기존의 `stage_group`, `stage_label`처럼 개발자식 컬럼명은 사용하지 않고, 보고용으로 이해하기 쉬운 한글 컬럼명만 사용한다.

최종 컬럼은 다음과 같다.

| 컬럼 | 설명 |
|---|---|
| 질문 번호 | 평가셋 질문 ID |
| 질문 | 사용자 질의 |
| 정답 텍스트 | testset.xlsx에 기록된 기준 답변 |
| 정답 이미지 | testset.xlsx에 기록된 기준 이미지 파일명 |
| 실습 단계 | 해당 질문이 속하는 실습 상황 또는 단계 |
| 질문 유형 | 질문의 성격 |
| 라벨링 근거 | 해당 실습 단계로 분류한 이유 |

## 실습 단계 예시

| 실습 단계 예시 | 포함 내용 |
|---|---|
| 안전/전원/케이블 | 케이블 개조, 전원 입력, 접지, 극성, 비상 복구 |
| 설치/기구 고정 | 베이스 고정, 마운팅, 케이블 배선 |
| 제어기 I/O 배선 | Digital I/O, Safety I/O, 센서 배선 |
| 툴 플랜지/I/O 배선 | 엔드 이펙터 전원, 플랜지 통신 핀 |
| 티치 펜던트/상태 확인 | 화면 메뉴, 비상정지 버튼, USB 포트 |
| TCP/툴 좌표 설정 | TCP offset, Tool Coordinate |
| 안전 설정/Zone 형상 | 안전 영역, 협착 방지, 공간 제한 구역 |
| 프로그래밍/모션 명령 | Task Editor, MoveL, MoveJ, Home Position |
| 시스템 관리/로그 | 로그, 업데이트, 공장 초기화, 사용자 권한 |

## 산출 파일

- `excel/01_question_stage_labels.xlsx`: 교수님 보고 및 직접 확인용 엑셀 파일
- `data/01_question_stage_labels.csv`: 동일 내용을 담은 원본 데이터 파일

엑셀에서 한글이 깨질 수 있으므로 보고와 수정은 `.xlsx` 파일을 기준으로 한다.
